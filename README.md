# GasAja! 🚀

**Platform Hangout Planning & Social Network Gen Z Paling Seru!**

[![Vite Build](https://img.shields.io/badge/Vite-Build-00FF9F?style=for-the-badge&logo=vite&logoColor=0a0a0a)](https://vite.dev/)
[![React 19](https://img.shields.io/badge/React-19-blue?style=for-the-badge&logo=react)](https://react.dev/)
[![TanStack Router](https://img.shields.io/badge/TanStack-Router-FF4D00?style=for-the-badge&logo=react)](https://tanstack.com/router)
[![Firebase](https://img.shields.io/badge/Firebase-Realtime-FFCA28?style=for-the-badge&logo=firebase)](https://firebase.google.com/)

**GasAja!** adalah platform hybrid sosial + *intelligent planning* yang dibuat khusus untuk Gen Z Indonesia agar bisa hangout secara spontan tapi tetap terorganisir. Gabungan kekuatan Instagram feed, WhatsApp coordination, dan Google Calendar dalam satu platform yang estetik, super cepat, dan real-time.

---

## 🎯 Value Proposition

> **"Karena hidup terlalu pendek untuk sekadar berdebat 'Mau nongkrong kemana ges?' di group chat."**

GasAja! memecahkan masalah klasik overthinking dan kelamaan diskusi saat mau main bareng. Cukup bikin Plan dalam 3 langkah mudah, bagikan link, dan biarkan teman-temanmu gabung (*join*) secara instan.

---

## ✨ Fitur Unggulan

- **🏠 Home Feed (Recommendation Engine)**: Infinite scroll estetik yang menggabungkan Post harian dengan Plan aktif menggunakan algoritma rekomendasi *discovery-driven* (skor berdasarkan engagement, aktualitas, media, dan variasi).
- **📅 3-Step Plan Wizard**: Buat rencana nongkrong instan lengkap dengan Judul, Tanggal, Jam, Pilihan Peta (Mapbox GL), Max Slot Orang, serta Vibe (Chill ☕, Aesthetic ✨, Party 🎉, dll.).
- **👥 Social Collaboration**: Follow teman, ketahui status kehadiran (*Joined*), kirim komentar, simpan plan ke koleksi personal, dan share ke media sosial.
- **📍 Location Intelligence**: Geolocation real-time bertenaga Mapbox Geocoding untuk mencari tempat nongkrong, *hidden gems*, cafe, mall, hingga tempat olahraga terdekat.
- **🛎️ Real-Time Notifications**: Notifikasi instan menggunakan Firebase Realtime Database saat ada yang menyukai post, berkomentar, atau gabung ke plan kamu.
- **📖 Instant Caching & Offline State**: Memanfaatkan Zustand store untuk memuat feed secara kilat (*cache-first render*) tanpa *blank page* atau *loading spinner* yang lama.

## 🚀 Cara Install & Menjalankan Proyek Secara Lokal

### Prasyarat
Pastikan kamu sudah menginstal [Node.js](https://nodejs.org/) (versi 18 ke atas disarankan).

### Langkah-langkah:

1. **Clone Repository**:
   ```bash
   git clone https://github.com/username/gasaja.git
   cd gasaja
   ```

2. **Instal Dependensi**:
   ```bash
   npm install
   ```

3. **Konfigurasi Variabel Lingkungan (`.env`)**:
   Salin `.env.example` menjadi `.env` dan masukkan API Key Firebase & Mapbox milikmu:
   ```bash
   cp .env.example .env
   ```

4. **Jalankan Server Development**:
   ```bash
   npm run dev
   ```
   Buka [http://localhost:5173](http://localhost:5173) di browsermu!

5. **Lakukan Build untuk Produksi**:
   ```bash
   npm run build
   ```
